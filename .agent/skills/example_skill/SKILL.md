# Example Skill

This is a template skill layout. You can use this structure as a base for creating new skills.

## Structure

A skill folder (like this one) typically contains:

-   `SKILL.md` (Required): This file. Contains the instructions for the agent.
-   `scripts/` (Optional): A directory for any scripts (Python, Shell, etc.) that the skill needs to run.
-   `resources/` (Optional): A directory for static files, templates, or data files.
-   `examples/` (Optional): A directory for example files or code snippets.

## Instructions

When this skill is invoked or relevant, the agent will read these instructions.

1.  Describe the steps the agent should follow.
2.  Reference files in the `resources/` or `scripts/` directories using relative paths if needed.
